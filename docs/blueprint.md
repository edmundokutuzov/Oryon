# **App Name**: Enterprise Command

## Core Features:

- Login Screen: Secure login screen with corporate email and password fields.
- Dashboard Overview: Display key metrics, tasks, meetings, and recent activities.
- Task Management: Create, assign, and track tasks with deadlines and priorities.
- Meeting Scheduler: Schedule meetings, send invites, and manage participants.
- Department Overview: Summarize departments and current related project to increase organizational transparency
- Calendar Integration: Visualize schedules with team meetings and other events
- AI Assistant: Context-aware assistant tool that can help summarize the contents of any currently viewed resources.

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5) to convey trust, security, and professionalism.
- Background color: Dark slate gray (#1E293B) for a modern, sophisticated, and secure feel.
- Accent color: Teal (#06B6D4) to highlight interactive elements and key information with a sense of calm and clarity.
- Body and headline font: 'Inter' (sans-serif) for a clean, readable, and modern appearance.
- Font Awesome icons for a consistent, scalable, and recognizable visual language.
- Dashboard-style layout for quick access to key metrics and functionalities.
- Subtle fade-in effects and transitions for a smooth user experience.
